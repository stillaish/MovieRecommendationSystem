# movie-recommendation-system-using-content-based-filtering

A simple AI-powered **Movie Recommendation System** built using **Python, Pandas, and Scikit-learn**.

Users can search for a movie and instantly get **top 5 similar movie recommendations** based on title and genres.

---

## 🚀 Features

* 🔍 Search movies using partial names
* 🎯 Select movie from suggested list
* 🎬 Get top 5 similar movie recommendations
* ⚡ Fast similarity computation using cosine similarity
* 🧠 Uses NLP (CountVectorizer)
* 💻 Simple command-line interface

---

## 🛠 Tech Stack

* Python
* Pandas
* Scikit-learn

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/movie-recommender.git
cd movie-recommender
```

---

### 2️⃣ Install Dependencies

```
pip install pandas scikit-learn
```

---

### 3️⃣ Download Dataset

```
wget https://files.grouplens.org/datasets/movielens/ml-latest-small.zip
unzip ml-latest-small.zip
```

---

### 4️⃣ Run the Application

```
python app.py
```

---

## 📸 How It Works

1. Enter part of a movie name
2. System shows matching movies
3. Select a movie using number
4. System recommends similar movies
